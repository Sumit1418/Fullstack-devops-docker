# Devops-FullStack-App

A production style full-stack application built using **React, Spring Boot, MySQL, and Docker**

## Tech Stack

- Frontend: React + Nginx
- Backend: Spring Boot (REST APIs)
- Database: MySQL
- Containerization: Docker & Docker Compose
- ORM: JPA / Hibernate
- API Testing: Postman

## Architecture

Browser -> React(Nginx) -> Spring Boot -> MySQL

## Features

- User Registration
- User Login
- Secure Password Handling
- Database Persistence using Docker Volumes
- Environment-based Configuration
- Fully Dockerized Setup

## Run Locally (Docker)

```bash
git clone <repo-url>
cd Devops-FullStack-App
docker compose up --build
```
